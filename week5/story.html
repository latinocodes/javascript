Almost Killed my Heating System

This story talks about how I almost damage my house heating system.

I started messing around with creating my own IoT (Internet of things). This is one of the many projects I'm working on and it is to automate my heating system at home.

For this I'm using a few temparature sensors connected to a microcontroller (Anduino Uno), which these are them connected to a Raspberry PI.

There are 6 microcontrollers and 2 RaspberryPI in total of which 3 are connected to one Raspberry PI

RaspberryPI are then connected remotely to linux server in my house that's hosting the database and the API for the web interface I'm also building.

To make the story short, because I was testing the device a lot with the actual heating switch, I almost burned down the entire system for turning on and off too quick and obten.

My recommandation is to not do this at all, lol.
